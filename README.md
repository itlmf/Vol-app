# تطبيق المتطوعين - LifeMakers Egypt

## نظرة عامة

تطبيق شامل لإدارة أنشطة المتطوعين والفعاليات لمنظمة صناع الحياة مصر. يوفر التطبيق منصة متكاملة للمتطوعين للتفاعل والمشاركة في الفعاليات الاجتماعية والبيئية والتعليمية.

## المميزات الرئيسية

### 🏠 الصفحة الرئيسية (Home)
- **إجراءات سريعة**: إنشاء منشورات جديدة، فعاليات، رفع شهادات
- **تغذية الأخبار**: عرض المنشورات والإنجازات والفعاليات
- **تصفية المحتوى**: تصفية حسب النوع (إعلانات، إنجازات، فعاليات)
- **أدوات الإدارة**: أدوات خاصة للمشرفين والمديرين

### 📅 الأحداث (Events)
- **عرض الفعاليات**: قائمة شاملة بالفعاليات المتاحة
- **التسجيل**: تسجيل مباشر في الفعاليات
- **QR Check-in**: تسجيل الحضور عبر رمز QR
- **التصفية**: تصفية حسب النوع والموقع

### 🎁 المكافآت (Rewards)
- **نظام النقاط**: جمع النقاط من المشاركة في الفعاليات
- **استبدال المكافآت**: استبدال النقاط بمكافآت حقيقية
- **المستويات**: نظام مستويات متدرج (برونزي، فضي، ذهبي، بلاتيني)
- **التقدم**: عرض التقدم نحو المستوى التالي

### 👥 المجتمع (Community)
- **المجموعات**: انضمام لمجموعات حسب المحافظة أو الاهتمام
- **الأصدقاء**: إضافة أصدقاء ومتابعة نشاطاتهم
- **التواصل**: محادثات جماعية ومشاركة الصور
- **الأنشطة**: عرض آخر النشاطات في المجتمع

### ⚙️ الإعدادات (Settings)
- **الملف الشخصي**: تعديل البيانات الشخصية والصورة
- **الخصوصية**: إعدادات الخصوصية والأمان
- **الإشعارات**: التحكم في الإشعارات
- **المظهر**: تبديل بين الوضع الفاتح والداكن

## نظام الأدوار والصلاحيات

### المستخدم العادي (User)
- إنشاء منشورات شخصية
- التسجيل في الفعاليات
- التفاعل مع المحتوى (إعجاب، تعليق، مشاركة)
- عرض الملف الشخصي والإنجازات

### المشرف (Moderator)
- مراجعة وحذف المنشورات غير المناسبة
- إدارة التقارير
- عرض جميع المنشورات

### المدير (Admin)
- الموافقة على المنشورات الجديدة
- إدارة المستخدمين
- إنشاء وإدارة الفعاليات
- إصدار الشهادات

### المدير الأعلى (SuperAdmin)
- إدارة جميع الأدوار
- حظر وإلغاء حظر المستخدمين
- إدارة النظام بالكامل
- تغيير صلاحيات المستخدمين

## التثبيت والتشغيل

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- دعم JavaScript
- اتصال بالإنترنت (للتحميل الأولي)

### التثبيت
1. قم بتحميل الملفات
2. افتح `index.html` في المتصفح
3. استمتع باستخدام التطبيق!

2. اضغط على "Add to Home Screen"
